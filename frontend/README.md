# 🩺 CareGuru+

An AI-powered healthcare assistant built with **React, TypeScript, Express, Prisma, and PostgreSQL**. CareGuru+ helps users perform preliminary symptom assessments through an intelligent conversational interface while following safe medical guidance practices.

---

## Features

* 🤖 AI-powered health conversations
* 🩺 Symptom assessment workflow
* 🚨 Emergency symptom detection
* 📋 Structured diagnosis cards
* 💊 Action recommendations
* 💬 Conversation history
* 🎨 Modern responsive UI
* 🔐 Clean Architecture backend
* 🗄️ PostgreSQL + Prisma integration

---
